Lightning Module

This module allows to atomically and trustlessly (in the federated trust model) interact with the Lightning network through a Lightning gateway. See LightningModule for a high level overview.

Attention: only one operation per contract and round

If this module is active the consensus’ conflict filter must ensure that at most one operation (spend, funding) happens per contract per round
